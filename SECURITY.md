# Security Policy

## Supported Versions

At this time, only the latest release of AEGIS receives security updates.

| Version        | Supported |
| -------------- | --------- |
| Latest Release | ✅ Yes     |
| Older Releases | ❌ No      |

## Reporting a Vulnerability

If you discover a security vulnerability in AEGIS, please do **not** disclose it publicly before it has been reviewed.

Instead, please:

* Open a private security report if GitHub Security Advisories are enabled, or
* Contact the project maintainer with a detailed description of the issue.

Please include:

* A description of the vulnerability
* Steps to reproduce the issue
* The affected version
* Any relevant logs, screenshots, or proof of concept

## Responsible Disclosure

Please allow reasonable time for the vulnerability to be investigated and, if confirmed, fixed before making any public disclosure.

Thank you for helping keep AEGIS secure.
